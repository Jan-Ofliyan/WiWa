# WiWa — Wisdom Waves

Нейромузыкальный сервис для управления состояниями мозга через бинауральные биения.
Аналог Brain.fm, полностью на фронтенде — без сервера, без внешних аудиофайлов.

---

## Стек

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS v4** (синтаксис `@import "tailwindcss"`, `@theme inline`)
- **Web Audio API** — весь звук генерируется в браузере
- **Framer Motion** (зарезервирован, пока не используется)

---

## Концепция

Пользователь выбирает режим, длительность и интенсивность → нажимает Play →
браузер генерирует бинауральные биения через Web Audio API.

**Бинауральные биения**: левое ухо слышит несущую частоту (carrier), правое — carrier + beat.
Разница между ними = целевая частота мозговых волн. Нужны наушники.

---

## Режимы

| Режим | Тип волн | Биение | Несущая | Состояние |
|-------|----------|--------|---------|-----------|
| Фокус | Бета | 16 Гц | 200 Гц | Ясность, концентрация, рабочий поток |
| Медитация | Альфа | 10 Гц | 180 Гц | Спокойствие без сонливости |
| Тишина | Тета | 6 Гц | 160 Гц | Ворота в подсознание, творчество, интуиция |
| Сон | Дельта | 2 Гц | 150 Гц | Глубокое восстановление, глимфатическая система |
| Гамма | Гамма | 40 Гц | 200 Гц | Пиковая осознанность, когерентность мозга |

## Архитектура звука (`useAudioEngine.ts`)

```
AudioContext
├── ChannelMerger(2)
│   ├── ch.0 — leftOsc (carrierFreq) → leftGain(0.5)
│   └── ch.1 — rightOsc (carrierFreq + beatFreq) → rightGain(0.5)
├── noiseSource (AudioBufferSourceNode, loop) → noiseGain
└── masterGain → destination
```

- **Fade in/out**: 3 секунды (`FADE_DURATION = 3`)
- **Интенсивность**: `light 0.3 / medium 0.6 / deep 1.0` × masterGain
- **Розовый шум**: 2-секундный буфер, алгоритм b0–b6
- **Важный фикс**: при stopNodes() сначала захватываем refs в локальные переменные, обнуляем refs, потом через setTimeout останавливаем — иначе stale closure убивает новые осцилляторы

## Параметры плеера

- **Длительность**: 15 / 25 / 45 / 60 минут
- **Интенсивность**: Лёгкая / Средняя / Глубокая
- **Громкость**: 0–100%
- Таймер обратного отсчёта + прогресс-бар
- Подсказка "надень наушники" появляется на 4 сек при первом старте
